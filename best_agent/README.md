<h1 align="center">
    The best-belief agent discovered by the Huxley-Gödel Machine
</h1>


This folder contains the best-belief agent discovered by the Huxley-Gödel Machine as done in Section 4.3 of the paper. To evaluate it on SWE-bench Lite, run
```bash
PYTHONPATH=. python evaluate_agent.py --agent_path best_agent
```

Check the results [here](https://github.com/Wenyi-AI-Wang/swe-bench-experiments/tree/hgm/evaluation/lite/20251030_HGM_gpt5).
