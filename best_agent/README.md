<h1 align="center">
    The best-belief agent discovered by Huxley-Gödel Machine
</h1>


This forder contains the best-belief agent discovered by Huxley-Gödel Machine as done in Section 4.3 of the paper. To evaluate it on SWE-bench Lite, run
```bash
PYTHONPATH=. python evaluate_agent.py --agent_path best_agent
```
